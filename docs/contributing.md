# 🦊コントリビュート🐔
Frostへのコントリビュート方法について

## 基本的な開発の流れ
基本的には以下の流れで作業を進める。
1. 作業ブランチで作業する
2. 自分でdevelopにマージする

マージしてもよいか分からない場合は以下のようなフローで作業を進める。
1. 作業ブランチで作業を行う
2. PullRequestを立てる
3. 詳しい人にレビューしてもらう
4. 詳しい人(または自分)がdevelopにマージする

変更の方向性の議論が必要な場合は、issueまたはdiscussionを立ててから作業してください。
ただし、大規模の変更を行うなどの例外があれば、この方法に従う必要はありません。
